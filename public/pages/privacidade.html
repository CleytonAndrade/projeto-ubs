Politica de privacidade

Página Inicialhome Entrar/Cadastrar

1. Coleta de Informações

Coletamos informações fornecidas voluntariamente pelos usuários através dos formulários do site, como nome, e-mail, telefone, endereço e dados de login.

2. Uso das Informações

As informações coletadas são utilizadas exclusivamente para os seguintes fins:

  • Cadastro e autenticação de usuários;
  • Melhoria da experiência do sistema;
  • Contato com o usuário, quando necessário.

3. Compartilhamento de Dados

Não compartilhamos dados pessoais com terceiros, exceto quando exigido por lei.

4. Segurança

Adotamos medidas razoáveis para proteger os dados dos usuários contra acesso não autorizado, perda, uso indevido ou alteração.

5. Direitos do Usuário

Nos termos da Lei Geral de Proteção de Dados (LGPD), o usuário pode solicitar:

  • Confirmação da existência de tratamento de dados;
  • Correção de dados incompletos ou desatualizados;
  • Exclusão de dados pessoais, quando aplicável.

6. Uso de Cookies

Este site não utiliza cookies para rastreamento. Se no futuro isso mudar, os usuários serão devidamente informados.

7. Alterações nesta Política

Esta política pode ser atualizada a qualquer momento. Recomendamos que os usuários revisem esta página periodicamente.
